Student Performance Predictor API

API REST para predicción de rendimiento estudiantil usando Machine Learning

Version 1.0.0 Base URL: /api

Índice

1. Introducción 2. Autenticación 3. Endpoints 4. Modelos de Datos 5. Ejemplos de Uso 6. Códigos de Error

Introducción

Esta API proporciona acceso a un modelo de Machine Learning optimizado para predecir el rendimiento estudiantil basado en múltiples factores académicos, personales y socioeconómicos.

Modelos Disponibles: 3 modelos ML optimizados (SVR Ensemble, Gradient Boosting, Random Forest)
Optimización: Random Search con 50-100 iteraciones
Métricas del Mejor Modelo: R² = 0.8350 (Ensemble), RMSE = 6.05, MAE = 4.51

Autenticación

Versión actual: La API no requiere autenticación. En producción se recomienda implementar autenticación vía API Keys o JWT.

Predicción Individual

POST
/api/predict

Realiza una predicción de rendimiento estudiantil basada en los datos proporcionados.

Headers

Content-Type: application/json

Request Body

{
  "study_hours": 6,
  "attendance": 90,
  "sleep_hours": 8,
  "internet_usage": 3,
  "assignments_completed": 10,
  "previous_score": 75,
  "placement_status": "Placed"
}

⭐ Feature Engineering Automático

Las siguientes features se calculan automáticamente:

  • cognitive_efficiency = study_hours / sleep_hours
  • academic_engagement = (assignments_completed × attendance) / 100
  • previous_score_quartile: Q1(<50), Q2(50-65), Q3(65-80), Q4(≥80)

Parámetros

Campo Tipo Descripción Valores
study_hours float Horas de estudio diarias 0.0 - 24.0
attendance float Porcentaje de asistencia 0 - 100
sleep_hours float Horas de sueño 0.0 - 24.0
internet_usage float Horas de uso de internet 0.0 - 24.0
assignments_completed int Tareas completadas 0 - 20
previous_score float Calificación anterior (se categoriza en cuartiles) 0 - 100
placement_status string Estado de colocación laboral (opcional) Placed, Not Placed
model_type string Modelo de ML a usar (opcional, por defecto: SVR_Optimized) SVR_Optimized, GradientBoosting_Optimized, RandomForest_Optimized

Respuesta (200 OK)

{
  "prediction": 78.45,
  "confidence_interval": {
    "lower": 66.29,
    "upper": 90.61
  },
  "performance_category": {
    "category": "Bueno",
    "description": "Rendimiento satisfactorio",
    "alert_level": "info"
  },
  "model_used": "Ensemble_Voting",
  "model_r2": 0.8350,
  "timestamp": "2026-06-16T12:00:00"
}

Ejemplos de Uso

cURL

curl -X POST https://your-api.com/api/predict \
  -H "Content-Type: application/json" \
  -d '{
    "study_hours": 5.5,
    "attendance": 85,
    "previous_score": 72,
    "sleep_hours": 7,
    "class_participation": 4,
    "extracurricular": 1,
    "parental_involvement": "High",
    "motivation_level": "High",
    "internet_access": 1,
    "tutoring": 1,
    "family_income": "Medium",
    "teacher_quality": "High",
    "school_type": "Public",
    "peer_influence": "Positive",
    "learning_disabilities": 0,
    "gender": "Female"
  }'

Python (requests)

import requests

url = "https://your-api.com/api/predict"
data = {
    "study_hours": 5.5,
    "attendance": 85,
    "previous_score": 72,
    # ... resto de parámetros
}

response = requests.post(url, json=data)
result = response.json()
print(f"Predicción: {result['prediction']}")
print(f"Categoría: {result['performance_category']['category']}")

JavaScript (fetch)

const data = {
  study_hours: 5.5,
  attendance: 85,
  previous_score: 72,
  // ... resto de parámetros
};

fetch('https://your-api.com/api/predict', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify(data)
})
.then(response => response.json())
.then(result => {
  console.log('Predicción:', result.prediction);
  console.log('Categoría:', result.performance_category.category);
});

Códigos de Error

Código Descripción Solución
400 Bad Request - Datos inválidos Verificar formato JSON y campos requeridos
404 Not Found - Endpoint no existe Revisar URL del endpoint
500 Internal Server Error Contactar soporte técnico
503 Service Unavailable - Modelo no cargado Reiniciar servidor o reentrenar modelo
